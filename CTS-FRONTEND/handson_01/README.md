# Handson 01 - Student Portal

This folder contains the React Student Portal implementation for the frontend hands-on track.

## Run locally

```bash
npm install
npm run dev
```

## Contents

- `src/data/` for the portal course catalog
- `src/api/` for mock async data access
- `src/store/` for Redux Toolkit state and selectors
- `src/components/` for shared UI blocks
- `src/pages/` for route-based screens
